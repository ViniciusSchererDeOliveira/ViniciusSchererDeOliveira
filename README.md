#### 🏙️ Project UtopiA *(Closed Source)*
*A core simulation engine linked to the THESTRGZR ecosystem.* 

An autonomous, LLM-driven multi-agent simulator built with TypeScript, designed as an advanced testbed for AI ethical reasoning and alignment. Inspired by the *Smallville* experiment, it integrates the "Quantified Ethics" from the Aeonisk framework and the 6-tier counterfactual resolution mechanics from the YAGS engine. It relies on factual memory storage and custom LoRAs to ensure agents navigate morally complex scenarios with strict mechanical consequences, free from standard assistant biases.

#### 🔫 PULSE_RUNNER
*"Escape will make me God."* 

A text-based, LLM-driven extraction shooter inspired by the gritty, retro-futuristic aesthetic of games like Marathon. Built entirely with TypeScript to orchestrate complex AI models, allowing the system to dynamically generate the world and make subjective, real-time decisions based on player interaction.
